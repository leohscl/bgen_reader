# Bgenox

Rust version of the bgenix tool

Features:
- [x] samples reading from bgen
- [x] samples reading from sample file
- [x] iterator implementation for bgenstream
- [x] index file
- [x] list
- [x] incl-range
- [x] incl-rsid
- [x] excl-range
- [x] excl-rsid
- [x] genotype reading
- [x] vcf
- [x] test phased data
- [ ] file filters
